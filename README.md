# Extension Manager

Responsive web app built with React.js, Vite, TailwindCSS, Zustand, and TypeScript.

## Features

- Toggle between dark mode and light mode
- List all extensions with options to activate, deactivate, or remove
- Filter extensions by ALL, ACTIVE, or INACTIVE status

## Installation

1. Clone the repository

`git clone git@github.com:renato-vn/browser-extensions-manager.git`

2. Install dependencies

`yarn`

3. Run the development server

`yarn dev`

## Usage

Open the app in your browser. Use the dark/light mode toggle button at the top. Manage your extensions by activating, deactivating, or removing them. Use the filter buttons to show all, active, or inactive extensions.
